**Order Taking System for Sales Team**

A full-stack sales order management system designed to streamline distributor order collection and administrative operations for field sales teams.

**Overview**

This system enables sales representatives to digitally capture distributor orders in real time while providing administrators with centralized control, monitoring, and historical analytics through a dedicated admin panel.

The application improves order tracking, reduces manual errors, and provides better visibility into sales operations and distributor activity.

**Key Features**
